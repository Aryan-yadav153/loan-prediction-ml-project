Loan Prediction ML Project:-
This project predicts loan approval status using Machine Learning algorithms based on customer loan data.

Project Objective:-
The main objective is to analyze loan applicant data and predict whether a loan will be paid off or collection is needed.

dataset used:-
. loan_train.csv

Feature Used:-
1. Principal
2. Terms
3. Age
4. Gender
5. Education
6. Weekend

Data Preprocessing:-
1. Removed unncessary columns
2. Converted data columns
3. Encoded categorical values
4. Created Dummy variables
5. Standardized feature values

Machine Learning Models Used:-
1. K-Nearest Neighbour (KNN)
2. Decision tree
3. Support Vector Machine (SVM)
4. Logistic Regression

Process:-
1. Load dataset
2. Clean data
3. Feature selection
4. Train-test split
5. Train models
6. Compare accuracy

Output:-
The project comapares accuracy of multiple ML algorithms and shows which model performs better.

Libries Used:-
. pandas
. numpy
. matplotlib
. scilit-learn
